# JP_vocab
可以背诵单词或练习拼写，支持自主导入词库

当前版本支持所有语言的背诵/拼写，但是后续更新将着重考虑日语语法特性相关的适配
## 下载
下载安装包后根据安装向导完成安装（使用innosetup打包，暂未汉化安装程序，请见谅）

快捷方式的图标不能正常渲染，等待后续优化
## 使用
包含背诵（mode1）和拼写（mode2）两种模式

主页右上角支持导入单词文件，格式为csv格式，即`单词,释义`，分隔符为英文逗号
```
わたし(私),我
~さん,先生、女士、同志（称呼人时，为了表示礼貌接在名字后面的接尾词）
がくせい(学生),学生
かいしゃいん(会社員),公司职员
あなた(貴方),你、您
はい,是、对
いいえ,不、不是
```
### mode1
出现单词原文，选择认识/不认识，不认识的单词反复出现直到认识为止
### mode2
出现汉语释义，键入对应单词，检测标准会省略英文小中括号的内容以及日语波浪线 间隔号 日语省略号，即`(*)` `[*]` `～` `·`　`……`
## 注意
本软件区别于市面上的常规背单词软件的特性即是自主导入/自主添加，对大规模词汇的处理性能不足，望读者按需选择
